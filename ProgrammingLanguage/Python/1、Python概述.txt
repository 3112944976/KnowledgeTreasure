1、基本概念
Python是一种胶水语言，且不仅仅是脚本语言。最终我们可以说Python是一种免费、开源（IDE和代码均开源）、跨平台（可跨操作系统）的解释型高级动态（1>代码边编写，解释器边编译2>数据类型是基于值的动态储存原理的，其类型动态可变，无需随时去声明（可细分为变量类型、函数、类的动态（函数及类的动态性指当他们已经被定义好后，仍然可以修改其内部结构））编程语言。其也支持命令式和函数式的编程方式，同样也支持面向对象的程序设计（详见：6、类（面向对象的程序设计））。
2、编程规范
（1）注意缩进（其可体现出语句间的逻辑关系）
（2）不要写过长的语句。可以空格截断或者用（）把多行语句表现为一条语句。
（3）注意运算符、字符、代码模块的松散度与可读性
（4）代码的关键处，应注释其原理及作用
（5）有目的性的选择适合的数据类型，提高程序的运行速度。
（6）减少内循环中的无关计算，尽量往外提。
（7）巧妙利用运算符、生成器对象的惰性求值的原理，降低时空复杂度。
（8）在循环中应尽量引用局部变量，减少程序运行的时间。

4、库的导入
（1）import 模块名 as 别名
	使用该方法导入的库时，必须要以“模块名.对象名”的形式来访问使用。
（2）from 模块名 import 对象名 as 别名
	使用该方法导入的库时，不需要以“模块名.对象名”的形式来访问使用。
（3）from 模块名 import *
	导入指定模块的全部方法。
